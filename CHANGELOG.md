# Changelog

## 1.0.0

- Initial production-ready Windows QC benchmark launcher
- MAS-style terminal interface with single-key menu navigation
- Standalone EXE packaging through PyInstaller
- One-line PowerShell bootstrap that downloads and runs the release EXE
- Modular CPU, GPU, storage, battery, network, display, and thermal checks
- Self-contained HTML and JSON report generation
- GitHub Actions workflow for Windows EXE builds and tagged release assets
